# java-lotto-precourse

## 범위
- 입력
    1) 구입 금액(1,000원 단위, 양의 정수)
    2) 당첨 번호(쉼표 구분 6개, 1~45, 중복 없음)
    3) 보너스 번호(1~45, 당첨 번호와 중복 금지)
- 출력
    - `N개를 구매했습니다.` + 발행 로또 번호 **오름차순**
    - `당첨 통계` + `---`
    - 5등~1등 건수 및 **총 수익률(소수점 1자리, 둘째 자리 반올림)**

## 기능 목록
1. 구입 금액 입력/검증(양의 정수 & 1000 배수) — 오류 시 `[ERROR]` + 재입력
2. 로또 발행(N=금액/1000) — 각 티켓 1~45 중복 없는 6개
3. 발행 내역 출력 — `N개를 구매했습니다.` + 각 티켓 오름차순 출력
4. 당첨 번호 입력/검증 — 6개, 1~45, 중복 없음
5. 보너스 번호 입력/검증 — 1~45, 당첨 번호와 중복 금지
6. 등수 계산 — 1등(6), 2등(5+보너스), 3등(5), 4등(4), 5등(3)
7. 통계/수익률 출력 — 등수별 건수 & 총 수익률(둘째 자리 반올림 → 1자리)

## 구현 결정(Assumptions)
- 출력 포맷은 예시와 공백/개행까지 동일
- 상수는 전부 static final로 관리(가격/범위/상금/문구)
- UI(System.in/out) 로직은 단위 테스트에서 제외
